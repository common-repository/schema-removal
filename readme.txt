Remove Genesis theme Schema.org structured data markup from your WordPress site.

== Description ==

Genesis theme schema markup is great and super useful, however in some cases you will want to remove schema markups from your site, for example: when you want to implement schema by a plugin.

The Genesis Schema Removal plugin is what you need to do so.

== Installation ==

1. Upload the entire `schema-removal` folder to the `/wp-content/plugins/` directory.
2. DO NOT change the name of the `schema` folder.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. That is all.
